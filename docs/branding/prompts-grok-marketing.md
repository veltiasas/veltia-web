## TIPS PARA MEJORES RESULTADOS EN GROK

1. Si el resultado no tiene los colores exactos, agrega: "Use EXACTLY these colors: navy #003366, gold #FFD700, silver #C0C0C0"
2. Si agrega fotos no deseadas: "No photographs, no people, no stock photos. Abstract geometric illustration only"
3. Si el texto sale mal: "The text must read exactly: VELTIA" (Grok/Flux a veces genera texto imperfecto)
4. Para iterar: "Keep the same layout but change the headline to: [nuevo texto]"
5. Si queres variantes: "Generate 4 variations of this design with slightly different geometric patterns"
6. Para mas minimalismo: "Make it more minimal. Remove decorative elements. More whitespace"
7. Si sale muy cargado: "Simplify. Maximum 3 visual elements. Clean Swiss design style"
