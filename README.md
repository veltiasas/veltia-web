# VELTIA - Tecnologia que transforma

## Resumen

Empresa IT con base en Cordoba, Argentina. Servicios de transformacion digital, IA, cloud optimization y consultoria con impacto ambiental medible.

**Sitio web:** [veltia.ar](https://veltia.ar)
**Estado:** Operativa
**Estructura:** SAS (Sociedad por Acciones Simplificada)

## Servicios

1. Transformacion Digital
2. IA & Agentes Inteligentes
3. Modernizacion Legacy
4. Optimizacion Cloud & FinOps
5. Consultoria Profesional

## Estructura del Proyecto

```
docs/
├── legal/              # Documentacion legal y societaria
├── negocio/            # Estrategia y modelo de negocio
├── comercial/          # Material comercial
├── branding/           # Guia de marca
└── operativo/          # Procedimientos y checklists

web/                    # Next.js 16 + React 19 + Tailwind v4
├── src/app/            # Pages y layout
├── src/components/     # Componentes React
├── src/app/export/     # Export pages para Instagram/LinkedIn
└── public/             # Assets estaticos

plantillas/             # Templates de propuestas
```

## Stack Tecnologico

- **Frontend:** Next.js 16, React 19, TypeScript, Tailwind CSS v4
- **Animaciones:** Framer Motion
- **Iconos:** Lucide React
- **Deploy:** Cloudflare Pages (static export)

## Desarrollo

```bash
cd web
npm install
npm run dev      # Dev server
npm run build    # Static export a out/
```
